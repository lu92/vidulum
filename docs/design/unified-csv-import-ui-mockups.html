API Endpoints Used

Step Endpoint Purpose Key Response Fields
1. Upload POST /api/v1/csv-import/upload Upload CSV, auto-detect format (canonical/cached/AI) transformationId, detectionResult, suggestedStartPeriod, detectedBank, detectedCurrency
2. Create CashFlow POST /cash-flow/with-history Create CashFlow in SETUP mode with historical periods cashFlowId (string)
3a. Import to Staging POST /api/v1/bank-data-adapter/{transformationId}/import Send transformed CSV to staging for category mapping stagingSessionId, importedRows
3b. Get Unmapped Categories GET /api/v1/bank-data-ingestion/cf={id}/staging/{sessionId} Get list of bank categories that need mapping unmappedCategories[], status
3c. Create Mappings POST /api/v1/bank-data-ingestion/cf={id}/mappings Map bank categories to CashFlow categories mappedCount
3d. Revalidate POST /api/v1/bank-data-ingestion/cf={id}/staging/{sessionId}/revalidate Verify all categories are mapped status, stillUnmappedCategories[]
3e. Execute Import POST /api/v1/bank-data-ingestion/cf={id}/import Import transactions to CashFlow jobId, transactionsImported, categoriesCreated[]
4. Attest (Optional) POST /cash-flow/cf={id}/attest-historical-import Confirm balance and transition to OPEN status status: "OPEN"
