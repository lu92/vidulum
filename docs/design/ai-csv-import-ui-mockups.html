Error Scenarios

Obsługa błędów i edge cases

ERROR: AI_ADAPTER_EMPTY_FILE (HTTP 400)

Pusty plik

Przesłany plik CSV jest pusty lub nie zawiera danych.

ERROR: AI_ADAPTER_FILE_TOO_LARGE (HTTP 400)

Plik za duży

Rozmiar pliku: 6.2MB

Maksymalny dozwolony rozmiar: 5MB

Wskazówka: Podziel plik na mniejsze części lub usuń niepotrzebne kolumny.

ERROR: AI_ADAPTER_INVALID_FILE_TYPE (HTTP 400)

Niewłaściwy format pliku

Wykryto: .pdf

Wymagany format: .csv

ERROR: AI_ADAPTER_UNRECOGNIZED_FORMAT (HTTP 400)

Nie udało się rozpoznać formatu

AI nie był w stanie zidentyfikować banku ani struktury pliku.

Wykryte nagłówki:

Column1, Column2, Data, Wartość
ERROR: AI_ADAPTER_AI_SERVICE_UNAVAILABLE (HTTP 503)

Serwis AI niedostępny

Tymczasowy problem z usługą AI. Spróbuj ponownie za chwilę.

ERROR: AI_ADAPTER_RATE_LIMIT_EXCEEDED (HTTP 429)

Zbyt wiele żądań

Przekroczono limit zapytań do AI. Poczekaj chwilę.

Spróbuj za: 60s
ERROR: AI_ADAPTER_DUPLICATE_FILE (HTTP 409)

Plik już przetworzony

Ten plik został już wcześniej przesłany i przetworzony.
